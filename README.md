# react-edu-tutorial

## Recommended knowledge
- javascript, HTML, CSS
- ES6 (basics)
- npm

## Building
- build with `$ npm run build`

## Tutorial
1. view index.html in browser
2. render <Message>
3. ...

## Debug
- window object `window.tutorial.debug`
- dynamically adding messages `window.addMessage({id: 3}, name: 'Message')`
